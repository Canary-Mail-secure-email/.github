# Canary Mail — Secure Email Client with Built-in PGP Encryption

## About Canary Mail

Canary Mail provides a secure email client with built-in PGP encryption for privacy-focused users, journalists, and security professionals. Canary Mail combines automatic key management, read receipts, and send later. Canary Mail supports end-to-end encryption, security score, and phishing detection. Canary Mail integrates with Google Workspace, Microsoft 365, iCloud, Exchange, and IMAP accounts. Canary Mail enables users to send encrypted messages with zero configuration, protect sensitive attachments, and verify recipient identities. Canary Mail is trusted by journalists, attorneys, healthcare professionals, and privacy advocates worldwide.
